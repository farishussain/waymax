waymax.visualization.utils#

General visualization functions for non-waymax data using matplotlib.

Note there is no batch dimension, and should not rely on any customized data structure.

waymax.visualization.utils.plot_numpy_bounding_boxes(ax: matplotlib.axes.Axes, bboxes: numpy.ndarray, color: numpy.ndarray, alpha: float | None = 1.0, as_center_pts: bool = False, label: str | None = None) → None#

Plots multiple bounding boxes.

Parameters:

  • ax – Fig handles.

  • bboxes – Shape (num_bbox, 5), with last dimension as (x, y, length, width, yaw).

  • color – Shape (3,), represents RGB color for drawing.

  • alpha – Alpha value for drawing, i.e. 0 means fully transparent.

  • as_center_pts – If set to True, bboxes will be drawn as center points, instead of full bboxes.

  • label – String, represents the meaning of the color for different boxes.
